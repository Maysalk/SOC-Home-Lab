# SOC Home Lab

## Overview

This project documents my journey of building a complete Security Operations Center (SOC) Home Lab.

The goal is to gain hands-on Blue Team experience by building a realistic SOC environment from scratch.

---

## Lab Components

- Ubuntu Server
- Wazuh SIEM
- Windows 11
- Kali Linux
- Sysmon
- OpenSSH
