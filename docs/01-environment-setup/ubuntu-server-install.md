# Ubuntu Server Virtual Machine

## Overview

An Ubuntu Server virtual machine was deployed to host the Security Information and Event Management (SIEM) platform used throughout this project. This server serves as the central component of the SOC Home Lab and is responsible for collecting, indexing, and analyzing security events generated by monitored endpoints.

Ubuntu Server was selected due to its stability, lightweight footprint, and strong compatibility with enterprise security solutions such as Wazuh.

---

## Purpose

The Ubuntu Server virtual machine is used to:

* Host the Wazuh platform.
* Collect endpoint security logs.
* Process and index security events.
* Provide centralized security monitoring.
* Support threat detection and incident investigation.
* Act as the primary management server within the lab environment.

## Installation Summary

An Ubuntu Server virtual machine was successfully installed using VMware Workstation Pro. Following the installation, the operating system was updated and prepared for deployment of the Wazuh platform.

The server was configured to operate as a dedicated security monitoring server within the lab environment.

---

## Server Management

The Ubuntu Server is administered remotely using SSH from the host machine.

This approach reflects common enterprise practices where Linux servers are managed remotely rather than through a local graphical interface. Remote administration simplifies management while reducing resource consumption on the server.

The Wazuh Dashboard is accessed through a web browser running on the host machine, while backend administration tasks are performed through SSH sessions.
