# Project Overview

## Overview

This project documents the design, implementation, and validation of a Security Operations Center (SOC) Home Lab built using virtual machines and open-source security tools.

The primary goal of this lab is to simulate a realistic enterprise environment where cyber attacks can be performed, detected, investigated, and documented. Throughout the project, each phase is carefully documented to demonstrate practical cybersecurity skills and provide a reproducible learning resource.

---

## Project Goals

* Build a realistic SOC Home Lab from scratch.
* Deploy a centralized SIEM platform using Wazuh.
* Monitor Windows endpoints using Sysmon.
* Simulate real-world cyber attacks from Kali Linux.
* Develop custom detection logic.
* Investigate security events using centralized logging.
* Practice incident response procedures.
* Build a professional cybersecurity portfolio.

---

## Project Documentation

This project is organized into multiple phases, each focusing on a different aspect of building and operating a SOC environment.

* Lab Objectives
* Environment Setup
* Network Configuration
* Endpoint Monitoring
* Attack Simulation
* Detection Engineering
* Incident Response
* Troubleshooting
* Future Improvements
