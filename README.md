#  Overview

This project documents the development of a complete Security Operations Center (SOC) Home Lab designed to simulate an enterprise security environment.

The lab is built from scratch using virtual machines and open-source security tools to practice Blue Team operations, endpoint monitoring, threat detection, attack simulation, and incident response.

Every stage of the project is documented to demonstrate both technical implementation and security analysis.

---

#  Objectives

* Build an isolated enterprise-style SOC lab.
* Deploy and configure Wazuh as the SIEM platform.
* Monitor Windows endpoints using Sysmon.
* Simulate realistic cyber attacks from Kali Linux.
* Analyze security events and Wazuh alerts.
* Map detections to the MITRE ATT&CK Framework.
* Develop practical incident response skills.
* Create a professional cybersecurity portfolio project.
