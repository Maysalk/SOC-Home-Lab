# Lab Objectives

## Purpose

The primary purpose of this SOC Home Lab is to gain practical, hands-on experience with Security Operations Center (SOC) technologies, workflows, and defensive security practices.

Rather than focusing solely on theory, this project emphasizes building, configuring, monitoring, attacking, detecting, and documenting a complete security environment.

---

## Learning Objectives

By completing this project, the following skills will be developed:

* Building an enterprise-style virtual security lab.
* Deploying and managing Wazuh.
* Configuring Windows endpoint monitoring.
* Collecting and analyzing Windows Security Event Logs.
* Deploying and configuring Sysmon.
* Performing attack simulations using Kali Linux.
* Mapping detections to the MITRE ATT&CK Framework.
* Creating custom detection rules.
* Conducting incident investigations.
* Producing professional technical documentation.

---

## Technical Objectives

The lab is designed to achieve the following technical goals:

* Create an isolated virtual environment.
* Configure secure networking between virtual machines.
* Centralize log collection.
* Validate endpoint connectivity.
* Generate attack telemetry.
* Detect malicious activity.
* Investigate alerts.
* Document findings.

---

## Project Outcome

Upon completion, this repository will demonstrate practical SOC Analyst skills, including security monitoring, detection engineering, incident response, and technical documentation suitable for a cybersecurity portfolio.
